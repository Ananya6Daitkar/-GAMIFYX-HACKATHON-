## üéÆ Key Features

### For Students
- **XP & Leveling System** - Earn XP for submissions, reach new levels
- **Achievements & Badges** - Unlock badges for milestones (First Commit, Code Master, Streak Warrior)
- **Leaderboard** - Compete with peers, see rankings
- **Focus Mode** - 25/45/60-minute sessions with sound notifications and XP rewards
- **AI Feedback** - Get instant code suggestions from Ollama AI
- **Streak Tracking** - Maintain daily coding streaks for motivation
- **Analytics Dashboard** - Track progress, skills, and learning journey

### For Teachers
- **Class Overview** - See all students, average XP, top performers
- **Student Analytics** - Detailed progress tracking per student
- **Auto-Grading** - Automatic scoring based on code quality metrics
- **GitHub Integration** - Direct GitHub repo analysis
- **Submission Tracking** - Monitor all student submissions
- **Performance Insights** - Identify struggling students early

### Technical Features
- **Real-time AI Analysis** - Ollama integration for code feedback
- **GitHub OAuth** - Seamless GitHub authentication
- **Auto-Grading Algorithm** - 6-dimensional code quality scoring:
  - Commit Message Quality (10 pts)
  - Commit Count (10 pts)
  - Lines Balance (15 pts)
  - Required Files (20 pts)
  - Folder Structure (25 pts)
  - README Quality (20 pts)
- **Real-time Notifications** - Instant alerts for submissions and feedback
- **Responsive Design** - Works on desktop, tablet, mobile
